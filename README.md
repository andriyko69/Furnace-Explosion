# Furnace Explosion

A dumb little NeoForge mod for Minecraft **1.21.1**.

Put **TNT** into a **regular furnace**, add fuel, and start "smelting" it.

Right before the smelting finishes...

**the furnace explodes.**

## Notes
- This mod is meant as a joke / troll mod and is not balanced in any way

## Disclaimer

Do not trust furnaces.
